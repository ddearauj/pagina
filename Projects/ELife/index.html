Elife

This is the eletronic life project from the book Eloquent JavaScript which the idea is to create a "world simulator". Right now, this world consists of two simple things: Walls (represented by the "#"), Critters (represented by the "o") and Wallfollowers (represented by the "~"). Critters move in a single direction until they hit a wall, then it will switch directions. Wallfollowers will always move next to a wall

Right now I've only added a couple of things. First of all I added the option to run it on my own page and the option for the user to create a custum map
